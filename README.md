# FLAIRR-TS Reproducibility

This repository contains the code and scripts used to reproduce the results of the FLAIRR-TS model as described in the paper "FLAIRR-TS: Forecasting LLM-Agents with Iterative Refinement and Retrieval for Time Series" [arXiv:2508.19279](https://arxiv.org/abs/2508.19279).
